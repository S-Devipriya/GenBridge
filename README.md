# GenBridge
A Digital Literacy Platform Demo for Senior Citizens

This application is developed for participation in Digital Skills to Succeed (DS2S) Asia Hackathon on "Experiential AI Learning for Real World Problem Solving". As it is a no-code/low-code hackathon, the app is coded mainly using AI-generated code, with adjustments as needed. The app's innovative ideas are from team discussion as well as discussion with mentors assigned during the hackathon. AI is merely used as a tool to bring the proposed solution to life.

Gemini 3 Flash was used for majority of AI coding.
